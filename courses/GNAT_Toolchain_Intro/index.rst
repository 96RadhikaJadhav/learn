Introduction to GNAT Toolchain
==============================

.. container:: content-description

    This course presents an introduction to the GNAT toolchain included in the
    GNAT Community 19 edition, including first steps to get started with the
    toolchain and some details on the project manager (GPRbuild) and the
    integrated development environment (GNAT Programming Studio).
